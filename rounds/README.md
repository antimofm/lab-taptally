# rounds/

Design-round previews. Each subfolder is a full self-contained copy of the TapTally landing page with one coherent set of changes applied, browseable from its own URL. See `~/.taptally/context/design-rounds.md` for the workflow.

Naming: `NN-YYYY-MM-DD-<slug>/` — sequential, never reused, date is round-creation date.

Old rounds stay live as visible history. Don't delete unless Antimo asks.
